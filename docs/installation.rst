Installation
============

From PyPI
---------

.. code-block:: bash

   pip install ncountr

With cross-platform validation support (adds scanpy and anndata):

.. code-block:: bash

   pip install ncountr[crossplatform]

From conda-forge
----------------

.. code-block:: bash

   conda install -c conda-forge ncountr

From source
-----------

.. code-block:: bash

   git clone https://github.com/princello/ncountr.git
   cd ncountr
   pip install -e ".[dev]"
