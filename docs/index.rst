ncountr
=======

**Python pipeline for Nanostring nCounter data analysis.**

11 R packages exist for nCounter data, but until now there was no Python
option.  ncountr fills that gap — parse RCC files, run QC, normalize, and
test for differential expression in a single config file or Python API.
